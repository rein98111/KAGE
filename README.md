# 🎹 KAGE - Rhythm Game Project_

![KAGE Status](https://img.shields.io/badge/Status-Developing-black?style=for-the-badge)
![Platform](https://img.shields.io/badge/Platform-Web%20%7C%20Minecraft%20Plugin-white?style=for-the-badge)

《KAGE》是一款專注於極簡高對比視覺美學與硬核譜面觀測的音樂節奏遊戲。本儲存庫包含了官方網站的前端源代碼，並作為玩家社群討論與譜面資料庫的展示中心。

🔗 **[點此前往官方網站 / VISIT WEBSITE](https://rein98111.github.io/KAGE/index.html)**

---

## ⚡ 遊戲核心特色 / Features

* **機能風視覺設計**：全面採用黑白高對比的科幻感 UI 介面，減少視覺干擾，專注於擊打節奏。
* **動態玩家討論區**：基於 GitHub Pages 靜態架構，物理連線 Google Sheets API 實現的免費動態論壇。
* **硬核譜面資料庫**：內建黑白譜面預覽功能，供玩家在網頁端快速瀏覽高難度神仙配置。

---

## 🛠️ 未來更新計畫 / Roadmap

目前遊戲與網頁端持續開發中，預計推進以下核心功能：

### 1. 遊戲核心模組 (Mod System)
- [ ] **HD (Hidden)**：物件接近判定點前物理隱藏，考驗背譜與動態視力。
- [ ] **HR (Hard Rock)**：Scroll Speed (sS) 速度大幅加快，挑戰反應極限。
- [ ] **HT (Half Time)**：歌曲速度與判定放慢，供新手玩家或拆解複雜配置練習。
- [ ] **AT (Auto Play)**：自動遊玩觀測模式，用於譜面測試、判定校正與純賞譜。

### 2. UI 介面優化
- [ ] 於遊玩畫面最下方導入 **打擊誤差毫秒條 (Early/Late 判定顯示)**。
- [ ] 遊戲Note物件美化。
